Textbooks and Online Resources

Textbooks

  • Guttag, J. V. (2021). Introduction to Computation and Programming Using Python, Third Edition. MIT Press. [Amazon link]
    • Provides us with an introduction to principles and concepts of Computer Science.
  • Stephenson, B. (2019). The Python Workbook, Second Edition. Springer International Publishing. [Amazon link]
    • Provides us with a number of exercises to be completed for homework and assignments.
  • Hogan, Brian P. (2015). Exercises for Programmers. [Amazon link]
    • Additional exercises.
  • Lutz, Mark (2013). Learning Python, 5th Edition. O'Reilly Media. [Amazon link]
    • Provides us with a very comprehensive guide to Python syntax and language features.

Webpages

  • How to Think Like a Computer Scientist: Learning with Python 3 [link]
