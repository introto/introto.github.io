1.001: Intro to CompSci using Python

Course Objectives

  • Provide an understanding of the role computation can play in solving problems.
  • Provide an understanding of the thinking process required for problem-solving in Computer Science.

Course Topics

  • For a list of course topics, please see the Table of Contents of the textbooks we use, Introduction to Computation and Programming Using Python.

Textbooks and Resources

Textbooks will be provided during our first session.

Books

  • Guttag, J. V. (2021). Introduction to Computation and Programming Using Python, Third Edition. MIT Press.
    • Provides us with an introduction to principles and concepts of Computer Science.
  • Hogan, Brian P. (2015). Exercises for Programmers.
    • Provides us with a number of exercises to be completed for homework and assignments.
  • Stephenson, B. (2019). The Python Workbook, Second Edition. Springer International Publishing.
    • Provides us with additional exercises.
  • Lutz, Mark (2013). Learning Python, 5th Edition. O'Reilly Media.
    • Provides us with a very comprehensive guide to Python syntax and language features.

Webpages

  • MIT's OpenCourseWare page for 6.0001 [link]
    • Provides us with exercises, in-class work, and problem sets. You are free to watch the freely provided lectures.
  • Python.org's Official Documents (includes tutorial, language manual and library reference) [link]
    • This page includes a tutorial, language manual and library reference which will prove useful during your independent studyining, and while completing tests and assignments.
